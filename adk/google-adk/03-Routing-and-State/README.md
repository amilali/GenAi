# Module 3: Routing and State

> [!TIP]
> This module introduces the core architectural concept of ADK: the Hierarchical Agent Tree.

## 🎯 Module Goal
Learn how to create a parent agent that routes conversations to specialized sub-agents based on their descriptions, and learn how to save data across turns using the Session State.

## 🚀 How to Run
```bash
python3 agent.py
```
*Try asking to "brainstorm" a trip, then ask to "plan" a trip to see the routing in action.*
